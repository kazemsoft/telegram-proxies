# لیست پروکسی MTProto برای تلگرام

لیست به‌روز پروکسی‌های MTProto برای کاربران تلگرام.

[English](README.md)

## راهنمای اضافه کردن پروکسی به تلگرام

### روش اول: کپی از کلیپ‌بورد (توصیه می‌شود)

۱. **باز کردن فایل پروکسی:**
   - به این مخزن بروید
   - فایل `mtproto.txt` را باز کنید

۲. **کپی کردن تمام لینک‌های پروکسی:**
   - تمام متن را انتخاب کنید (Ctrl+A یا Cmd+A)
   - آن را کپی کنید (Ctrl+C یا Cmd+C)

۳. **باز کردن تلگرام:**
   - تلگرام دسکتاپ یا موبایل را باز کنید

۴. **دسترسی به تنظیمات پروکسی:**
   - **دسکتاپ:** Settings → Advanced → Connection type → Use custom proxy
   - **موبایل (اندروید/iOS):** تنظیمات → داده و حافظه → تنظیمات پروکسی

۵. **اضافه کردن از کلیپ‌بورد:**
   - دکمه "افزودن پروکسی از کلیپ‌بورد" یا گزینه مشابه را پیدا کنید
   - تلگرام به‌طور خودکار تمام لینک‌های پروکسی را از کلیپ‌بورد شما تشخیص می‌دهد
   - تمام پروکسی‌ها به‌صورت یکجا اضافه می‌شوند

۶. **فعال‌سازی پروکسی:**
   - یکی از پروکسی‌های اضافه شده را انتخاب کنید
   - کلید پروکسی را روشن کنید
   - تلگرام به‌صورت خودکار سریع‌ترین پروکسی را تست کرده و به آن متصل می‌شود

### روش دوم: اضافه کردن تک‌تک لینک‌ها

۱. **یک لینک پروکسی را کپی کنید** از فایل `mtproto.txt` (مثال: `https://t.me/proxy?server=...`)

۲. **لینک را باز کنید:**
   - در مرورگر خود پیست کنید یا اگر از اپلیکیشن استفاده می‌کنید مستقیماً کلیک کنید

۳. **تأیید در تلگرام:**
   - تلگرام به‌صورت خودکار باز می‌شود و از شما تأیید می‌خواهد
   - روی "اتصال" یا "استفاده از پروکسی" کلیک کنید

## توجه

این لیست به‌طور منظم به‌روزرسانی می‌شود. برای دریافت آخرین پروکسی‌های فعال، مرتباً بررسی کنید.
