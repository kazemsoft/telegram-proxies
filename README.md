## How to Add Proxies to Telegram

### Method 1: Copy from Clipboard (Recommended)

1. **Open the proxy file:**
   - Navigate to this repository
   - Open `mtproto.txt` file

2. **Copy all proxy URLs:**
   - Select all text (Ctrl+A or Cmd+A)
   - Copy to clipboard (Ctrl+C or Cmd+C)

3. **Open Telegram:**
   - Open Telegram Desktop or Mobile app

4. **Access Proxy Settings:**
   - **Desktop:** Settings → Advanced → Connection type → Use custom proxy
   - **Mobile (Android/iOS):** Settings → Data and Storage → Proxy Settings

5. **Add from Clipboard:**
   - Look for "Add Proxy from Clipboard" or similar option
   - Telegram will automatically detect all proxy URLs from your clipboard
   - All proxies will be added at once

6. **Enable Proxy:**
   - Select one of the added proxies
   - Toggle the proxy switch to ON
   - Telegram will automatically test and connect to the fastest proxy

### Method 2: Add Individual Proxy Links

1. **Copy a single proxy URL** from `mtproto.txt` (e.g., `https://t.me/proxy?server=...`)

2. **Open the link:**
   - Paste in your browser or click directly if using a compatible app

3. **Confirm in Telegram:**
   - Telegram will open automatically and ask to confirm
   - Click "Connect" or "Use Proxy"
